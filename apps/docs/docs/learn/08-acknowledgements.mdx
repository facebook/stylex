## Additional thanks

Additionally, we’d like to acknowledge the influence of the following projects
and individuals:

- [Atomizer](https://acss.io/) and Thierry Koblentz for pioneering atomic CSS at
  scale.
- [React Native](https://reactnative.dev/) and Christopher Chedeau for
  developing the idea of
  [CSS-in-JS](https://blog.vjeux.com/2014/javascript/react-css-in-js-nationjs.html).
- [React Native for Web](https://necolas.github.io/react-native-web/) and
  Nicolas Gallagher for first generating conflict-free atomic CSS from
  CSS-in-JS.
- [Linaria](https://linaria.dev/) for demonstrating the use of CSS Custom
  Properties to drive the values of dynamic styles.
